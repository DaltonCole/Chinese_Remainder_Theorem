#  Chinese Remainder Theorem

##  Introduction

This program applies the Chineses Remainder Theorem to solve for x among a system of congruences

##  Example

x = 1 mod 3
x = 2 mod 5
x = 4 mod 7

Solve for x

NOTE: In this sence, = stands for congruence.


##  How to run:

###  Command

python3 chinese_remainder_theorm.py

###  Input

3
1 3
2 5
4 7 

###  Output
67
